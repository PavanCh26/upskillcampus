# Smart Home Automation System Using IoT

This project was developed as part of the Free Summer Internship in Embedded Systems and IoT conducted by upskill Campus, The IoT Academy, and UniConverge Technologies Pvt. Ltd. (UCT).

## Project Description
The Smart Home Automation System Using IoT enables remote monitoring and control of household appliances using an ESP8266 NodeMCU microcontroller, DHT11 temperature and humidity sensor, and a 2-channel relay module.

## Components Used
- ESP8266 NodeMCU
- DHT11 Sensor
- 2-Channel Relay Module
- Light Bulb
- Fan
- Wi-Fi Network

## Features
- Remote appliance control
- Temperature monitoring
- Humidity monitoring
- Wi-Fi based communication
- Low-cost IoT implementation
